Puedes usar la app web de Gemini, gemini.google.com, para potenciar tus ideas. Con un poco de su ayuda, puedes hacer lo siguiente:
Para usar la app web de Gemini en gemini.google.com, necesitas cumplir con uno de los requisitos siguientes:
Obtén más información sobre los idiomas y países en los que puedes usar la app web de Gemini.
Para usar Gemini en tu Cuenta de Google de institución educativa o trabajo, el administrador debe habilitar el acceso. Consulta cómo usar Gemini en tu Cuenta de Google de institución educativa o trabajo.
Nota: Para iniciar un chat nuevo, en la parte superior, presiona Nuevo chat . Descubre cómo encontrar y administrar tus chats recientes.
Nota: Obtén información para revisar y controlar los archivos que subes.
Los usuarios de Gemini Advanced pueden subir y analizar archivos en la app web de Gemini. Descubre cómo agregar archivos a tus instrucciones.
Si quieres revisar una instrucción, puedes editarla, y la app web de Gemini volverá a generar la respuesta.
Importante: Esta función no está disponible para respuestas que la app web de Gemini genera con extensiones.
Para algunas instrucciones, puedes revisar otras versiones que crea la app web de Gemini. Esta opción solo está disponible para la respuesta más reciente.
Importante: Esta función solo está disponible en chats de Gemini Advanced y para código de Python.
Puedes editar y ejecutar el código de Python de una respuesta directamente en la app web de Gemini. Gracias a esta función, puedes experimentar con el código, aprender cómo los cambios afectan el resultado final y verificar que todo funcione como debería.
Nota: También puedes editar código de Python sin ejecutarlo o al revés. Por ejemplo, puedes usar el botón Ejecutar código  en el código que proporciona la app web de Gemini para verificar su resultado.

Puedes solicitarles a las Apps con Gemini que usen fuentes educativas acreditadas, como libros de texto de OpenStax, para responder. 
Descubre cómo modificar respuestas, exportar respuestas y compartir tus chats desde las Aplicaciones con Gemini.
Las Apps con Gemini podrían usar tu ubicación para brindar respuestas más relevantes a tus instrucciones. Se usará tu ubicación general o, si lo permites, la ubicación precisa de tu dispositivo. Descubre cómo administrar tu ubicación. 
En la app web de Gemini, se muestra tu ubicación en la parte inferior del menú.
Puedes seleccionar "Actualizar ubicación" para permitir que Gemini use la ubicación precisa de tu dispositivo.
Nota: Si estás usando la app de Gemini para dispositivos móviles, consulta cómo administrar la ubicación en ella.
